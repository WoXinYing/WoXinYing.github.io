我和书（四）

发表于 2020-09-28 分类于 关于书

　　梦寐以求的大学生活终于来临了，终于能见识一下老师口中的——到了大学就没人管你了，你就可以好好玩儿了。确实，没人管了……于是自己也不管自己，大学四年，就在没日没夜的有声小说中过去了。
　　新的环境，还是新鲜了一阵的，刚开始总会有一些豪情壮志，什么入社团啦、健身啦，渐渐地，都成了笑话。整个人都变得毫无斗志，连奖学金都懒得申请了。于是便从小说中寻找慰藉，短暂地躲避一下现实。那段时间，耳机是从不离身的，几乎是24小时听书，饿了就戴着耳机下去吃点，困了就戴着耳机睡一会儿，一天又一天，一年又一年。
　　这段时间算是阅读量疯狂增长的时候。看了很多不同种类的书，玄幻自不必说，又对盗墓、仙侠、都市、王朝更迭、末世等都有涉猎。其中，对末世类是尤为喜欢的。现在想来，也或许是自己身无大志，懒病缠身，对小人物在末世中苦苦挣扎，最终某得一片安身之地，关上门，大家互相取暖，打开门，外边黄沙漫天的那种情节颇为喜爱与沉迷。自己努力总是累且枯燥的，而看别人努力，一次次在生死边缘徘徊总是说不出的刺激。
　　如今已经工作，已不再像大学那般肆无忌惮了，但偶尔还是会看看小说，倒不是还想逃避生活，总感觉，社畜一直两点一线，朝九晚五也不是长久之计，总需要一些东西稍稍的调节一下。虽然成了社畜，倒也还没忘记自己曾经想写一本小说，不管什么题材，不管是否发表，不管写的好坏，总希望能对这些年看的书有一个交代吧。

我和书（三）

发表于 2020-09-23 更新于 2020-09-28 分类于 关于书

　　日子过得很快，或悲或喜，或好或坏，转眼间已是高中。
　　高中啊，清闲的日子已消失地无影无踪。住校意味着告别一切电子产品，家里的黑白电视再也不用担心我去拍它了，也是从那时起，那台陪伴了我三年的旧手机便再也没打开过了。除了学习还是学习的日子里，有一个喜欢看小说漫画的同学，真好。
　　老师说，他对看小说的执着，就好像狗对屎一般，永远也戒不了。狗不挑屎的酸甜苦辣咸，他对小说也是如此，在我的印象里，他好像什么小说都看。所以，他说他已经读了几亿字了，这点我是坚信不疑的。那时他也不止看小说，每周都要买《知音漫客》，三年以来，从不曾断过。也不敢带回家，最后便全扔了。
　　晚上回到寝室，无聊之余便借漫画来看看。一周一更的漫画，剧情总是进展得极慢，每次看完总是意犹未尽。也是因此，我个人是极喜欢听人剧透的，听着别人讲故事主线的各种起起伏伏，各个人物的结局……说不出的爽快。无法忍受漫画如龟爬般的进度，便打起了看小说的主意。那时候两周放假一天，从放学坐上老妈的电动车，到第二天上学下电动车，手机几乎是不离手的。所以，觉也基本是不睡的，作业就更不用说了。好在成绩是极好的，因此老妈也就听之任之了，就是把眼睛给整坏了。
　　就这样，高中三年过去了，从第一本《斗破苍穹》，到“土豆三部曲”，到《仙逆》、《遮天》等等，这三年下来，我的阅读量妥妥地达到了一亿字了。本以为高三的暑假会是激情澎湃，令人难忘的。哪知道从考完最后一门那一刻起，整个人就变得无欲无求了般，对什么都提不起兴趣。之前的豪言壮语，定下的各种计划，都成了笑话。于是乎，整个暑假都用来看小说了，困了睡，醒了看，饿了吃，吃的时候也要看。由于看的太多，记不清都看了什么了。所以，那些日子都他喵的活到狗身上去了，我竟除了记得看了小说之外，再没任何记忆了，真是失败呢。
　　从吃了散伙饭之后，就没怎么和同学联系过了，也不知那位同学是否成了小说家呢？希望有一天能看到他写的书。

我和书（二）

发表于 2020-09-18 更新于 2020-09-28 分类于 关于书

　　一如既往地穷，电视依旧是黑白的，娱乐项目依旧匮乏，我就这样度过了小学无忧无虑的时光。
　　家中能看的纸质书都看得差不多了，剩下的都是不喜欢看的……作为一个活泼少年，自然是不喜看《红楼梦》这些的。初中的日子过的就颇为忙碌了，也算是起早贪黑，平日里也没什么机会看书，周末还要上辅导班，便只有寒暑假这大把的时间可以自行安排了。
　　说来也是奇怪，自上了初中开始，就不喜再去网吧了，可能是初中作业太多，没了代写这条财路了吧。寒假还好，可以走亲串友，暑假就比较难熬了。日子很长，且热得让人兴不起任何出去的意思，风扇吹出的风都是热的。如此天气，暑假作业自是不想做的，却又苦于老母亲的压迫，便只能躲到二楼，图的一时清净。
　　确是清净了，但无聊也是确确实实的，老爸的旧手机成了我的救命稻草。从上楼那一刻起，打开收音机，调到FM106.0HZ（南阳电视台故事广播），一听就是一天。楼下的老母亲以为我学了一天，晚上的日子也好过了起来，我可以看会儿电视，虽然黑白电视只能收到寥寥几个台，不过能看总是好的。
　　那台旧手机陪了我一个个夏天，具体都听了些什么，现在也记不得了，手机也被拿去换剪子换菜刀了，可听书的爱好却延续至今。如今再回去，故事广播已经不怎么放评书了，好在又出现了其他听评书的路子，依旧可以在夜深人静的时候，闭上眼，进入一个又一个光怪陆离的世界，目睹一个又一个跌宕起伏的人生。
　　高中是要住校的，日子更为忙碌了。早读加上晚自习，上学的日子里，再难找什么消遣了。所幸室友极喜欢看小说和漫画，每周都要买《知音漫客》、《星漫》之类。无聊之际，便借来看看。哪知这一看，便如打开了潘多拉魔盒般，开启了持续至今的，与网文的恩怨情仇，每每想狠心抛弃，却都又欲罢不能……

我和书（一）

发表于 2020-09-16 更新于 2020-09-28 分类于 关于书

　　受限于文学功底，只能为这篇文章起了这么个名字，厚颜说它朴实无华吧，哈哈。
　　……
　　想了五分钟，还是没想起来我看的第一本书叫什么名字。只记得是带拼音的，有刘胡兰、雷锋、白求恩他们的故事，好像还是盗版，老爸从城里回来，一下给我带了两本，当时可是高兴了好一阵的。
　　随着渐渐长大，在那个没什么娱乐项目的年代，看书成了为数不多的乐趣了。那时候家里没什么钱，好在有个大我八岁的姐姐，姐姐文采是极好的，书也是极多的，就从她那里“拿”书看。一开始就“拿”了两本书，名字记得很清楚，一本是《雾都孤儿》，一本是《鲁宾逊漂流记》。当时看着这两本不是很厚，应该很快就能看完，而且包装挺精致的，感觉是正版的书，也有想跟小伙伴儿炫耀的想法，现在想想，那时候家里那么穷，哪里买得起正版的书呀。
　　书倒是很快就看完了，里边的故事深深吸引了我，往后的日子里想起，总是回味无穷。看完却没有赶快还回去，我带着去学校炫耀去了，哈哈。果然，得意忘形之下，把书借给了小伙伴，还回来的时候就只剩下一本了，还皱巴巴的。怀着忐忑的心情，偷偷把书放了回去，一连好几天不敢和姐姐说话，姐姐那么爱看书的一个人儿，被发现了怕不是要被打死。好在她好像一直没有发现，我也一直没吃这顿打。后来兜兜转转搬家了好些次，那本书也在颠沛流离中找不到了。
　　进城上学不久，网吧如雨后春笋开始疯狂冒头了，那时候查的也不是很严，小伙伴就经常喊我过去，然而还是一如既往地穷，没有零花钱，每天帮别人写作业赚的钱也只能玩儿一个小时。回家还有大把的时间无事可做，看书又成了主要乐趣，现在想想，我没能成网瘾少年，大概就是因为我没钱吧，哈哈。
　　这个时候姐姐已经上高中了，在我们市最好的高中，爸妈又有了新的鞭策我的素材了。姐姐在学校的作文大赛里得了一等奖，奖品是二十本世界名著，这些书可是让我小学剩下的日子里过得颇为惬意的。《围城》、《堂吉诃德》、《巴黎圣母院》等都是这个时候看的，这么多年过去了，还能记得的，也只剩下书名了……现在想来，倒是颇为后悔没有跟姐姐问问该怎么看书，也是她不好，都不教我，要不是穷，我肯定要变成网瘾少年了。
　　小学时候家附近有个小书店，里边的书都挺旧的，看起来也不是啥正版书，但是漫画真多，我和小伙伴经常跑去看。《老夫子》、《阿衰》、《七龙珠》都是那时候看的。我性格比较内向，脸皮比较薄，一直只看不买也挺不好意思的，看着书架上书挺乱的，就拉着小伙伴一起整理起来了，一干就干到了天黑。老板也是个极好的人儿，看着我俩忙活了这么久，就送了我俩一人一本漫画，送他的是什么我不记得了，也不记得送我的是什么了，只记得后来的日子里挺宝贝这本漫画的，脏了烂了都不舍得丢，被亲戚家的熊孩子要走的时候，还伤心了好久呢。
　　这就是我和书的第一段缘分了，因为穷不得不结缘，也幸好是因为穷而结缘。

前言

发表于 2020-09-16

　　一直说想干这干那，但总是三天打鱼两天晒网的；
　　说要发展的兴趣爱好，三分钟热度之后，就再不想碰了；
　　从高中到现在，看过的小说也不少了，到头来什么也没落下，那些日子就像是虚度了一样；
　　由于想法比较跳脱，倒是总有些稀奇古怪的想法，但总是个念头，也没有什么更深入的思考。
　　前些天想了想，总这样也不是个办法，索性建个个人博客，把自己学习的经历、奇怪的想法等等记录在上边，以此作为督促和激励。
　　若能遇几个志同道合的朋友，也是人生一大幸事。
